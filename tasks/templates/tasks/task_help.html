{% trans "Jak działają zadania?" %}
{% trans "Moje" %}

{% blocktrans %}Zakładka zbiera zadania, które Cię bezpośrednio dotyczą: zadania które podjąłeś (jesteś właścicielem) oraz zadania, w których zadeklarowałeś pomoc (kliknąłeś „Chcę pomóc").{% endblocktrans %}

{% trans "Oczekujące" %}

{% blocktrans %}Zadania aktywne, które jeszcze nie mają właściciela lub mają właściciela, ale nie zebrały wystarczającego poparcia (różnica głosów „chcę pomóc" minus „nie robimy tego" wynosi mniej niż 2). Każdy może podjąć takie zadanie przyciskiem „Podejmij".{% endblocktrans %}

{% trans "W realizacji" %}

{% blocktrans %}Zadania, które mają właściciela i zebrały co najmniej 2 netto deklaracji pomocy (głosy „chcę pomóc" minus głosy przeciw ≥ 2). Właściciel może zrezygnować — zadanie wraca wtedy do oczekujących. Głosy ≤ −2 automatycznie odrzucają zadanie.{% endblocktrans %}

{% trans "Zakończone" %}

{% blocktrans %}Zadania zamknięte przez właściciela (ukończone lub anulowane) oraz zadania odrzucone przez społeczność (suma głosów ≤ −2). Zakończone zadanie można ponownie otworzyć.{% endblocktrans %}

{% trans "Deklaracja pomocy" %}

{% blocktrans %}Przycisk „Chcę pomóc" to deklaracja zaangażowania — nie lajk. Liczba osób które go kliknęły widoczna jest na kafelku zadania. Wystarczające poparcie (≥ 2 netto) jest warunkiem przejścia zadania do realizacji.{% endblocktrans %}
